Bem-vindo à API de Usuários e Posts

Esta é uma API REST criada com Node.js, Express, TypeScript e Prisma.

Rotas disponíveis:

  • GET /users - Lista todos os usuários com seus posts.
  • POST /users - Cria um novo usuário.
    { "name": "Maria", "email": "maria@example.com" }
  • GET /posts - Lista todos os posts com seus autores.
  • POST /posts - Cria um novo post.
    { "title": "Título", "content": "Texto", "authorId": 1 }

Use ferramentas como Postman ou Insomnia para testar as rotas POST.
